| [Dating Compatibility Embedding System](https://github.com/sjordan1975/dating-compatibility-finetuning) | Fine-tuned sentence transformer to predict relationship compatibility from dating profile text — teaching a model that compatibility ≠ textual similarity. Key finding: generic profile language ("I value honesty") limits learnable signal; further improvement requires more specific training data, not more engineering. <br><br>Python · SentenceTransformers · UMAP · HDBSCAN · Pydantic · Matplotlib · pytest |
